belajar regex
-------------
1. untuk pattern matching (pencocokan pola)
2. untuk checking (pola yang seharusnya sudah benar atau belum)
3. untuk pencarian dan penggantian string (searching dan replacing string)

3. harus terdapat simbol
4. minimal 8 karakter
